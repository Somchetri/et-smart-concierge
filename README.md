# ET Smart Concierge 🤖💼

An AI-powered financial assistant that personalizes the Economic Times (ET) ecosystem experience.

## 🚀 Overview

ET Smart Concierge acts as a unified AI layer that:

- Understands user financial behavior
- Recommends personalized financial products
- Guides users through decisions
- Continuously learns from interactions

## 🧠 Key Features

### 1. Smart Onboarding
- 3-minute conversational profiling
- Builds "Financial DNA"

### 2. Financial Life Navigator
- Tracks goals, income, and risk
- Detects financial gaps

### 3. Cross-Sell Intelligence
- Behavior-triggered recommendations
- Personalized nudges

### 4. Marketplace Assistant
- Conversational interface for:
  - Loans
  - Insurance
  - Investments

---

## 🏗️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- AI Layer: OpenAI API (or mock)

---

## ⚙️ Setup Instructions

### 1. Clone Repo

```bash
git clone https://github.com/your-username/et-smart-concierge.git
cd et-smart-concierge
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Server

```bash
node server/index.js
```

### 4. Open Frontend

Open:
```
client/index.html
```

---

## 🔌 API Endpoints

### POST /api/concierge/chat

Request:
```json
{
  "message": "I want to invest money"
}
```

Response:
```json
{
  "reply": "Based on your profile, consider mutual funds..."
}
```
